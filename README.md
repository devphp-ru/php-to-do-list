# ToDoList
Демонстрационный проект списка дел, задач, на PHP

Model View Controller  

Что такое MVC?

Термин был впервые введен в 1970-х годах Тригве Реенскаугом.

В основе этого заключается в разделении частей приложения на
* Данные и бизнес-правила, называемые Моделью
* Презентация, обычно в HTML, называемая Представлением
* Соединительный слой между этими двумя частями, называемый Контроллер
